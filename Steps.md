# 🚀 План разработки Simple Online Store (v2026)

## Этап 1: Безопасность и Контроль (Критично)
1. **Интеграция authMiddleware:** Применить проверку JWT ко всем мутирующим роутам (создание/редактирование/удаление).
2. **Активация adminMiddleware:** Ограничить доступ к админ-панели (только для роли ADMIN).
3. **Тестирование прав:** Проверка статус-кодов 401 (Unauthorized) и 403 (Forbidden).

## Этап 2: User CRUD и Данные
1. **Реализация методов:** Дописать `getAll`, `getOne`, `update`, `delete` в UserController.
2. **Валидация:** Внедрить библиотеку (Zod или express-validator) для проверки email, цен и обязательных полей.
3. **Конфиг:** Привести `index.js` к единому порту (7000) и создать `.env.example`.

## Этап 3: Контракт для Vue-фронтенда
1. **Swagger:** Подключить `swagger-ui-express` для описания API.
2. **Error Handling:** Унифицировать формат JSON-ответов об ошибках.
3. **Static:** Проверить корректность отдачи изображений для будущего отображения в карточках товаров.

## Этап 4: Профессиональный Backend
1. **Миграции:** Настроить `sequelize-cli` и перенести структуру БД в файлы миграций.
2. **Логирование:** Настроить запись ошибок в `logs/error.log`.
3. **Unit-тесты:** Добавить базовые тесты на AuthService (регистрация/логин).

## Этап 5: Frontend Start (Vue 3)
1. Инициализация проекта через Vite.
2. Настройка Axios interceptors для автоматической подстановки JWT-токена.
3. Создание базовой структуры (Store через Pinia, Роутинг).
