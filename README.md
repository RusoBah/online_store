# 🛒 Simple Online Store (Backend)
Это учебный проект интернет-магазина (Node.js + Express + PostgreSQL). Проект переведен на архитектуру **Controller-Router-Model** для обеспечения чистоты кода и легкой интеграции с Vue.js.

## 🚀 Текущий статус: Связь установлена
Маршрутизация настроена и протестирована. Сервер успешно принимает запросы и возвращает ответы. Следующий этап — полная замена текстовых заглушек на логику взаимодействия с БД через Sequelize.

## 🛣 API Эндпоинты (Актуальные)

Все запросы к API начинаются с префикса `/api`.

### 📦 Товары (`/api/product`)
| Метод  | Путь                | Описание                                  |
| :----- | :------------------ | :---------------------------------------- |
| **GET** | `/product/getall`    | Получить все товары из БД                 |
| **GET** | `/product/getone/:id`| Детальная информация о товаре             |
| **POST**| `/product/create`    | Добавление товара (обязательно: `image`)  |

### 📂 Категории и Бренды
| Метод  | Путь                | Описание                                  |
| :----- | :------------------ | :---------------------------------------- |
| **GET** | `/category/getall`   | Список категорий (из контроллера)         |
| **POST**| `/category/create`   | Создать новую категорию                   |
| **GET** | `/brand/getall`      | Список всех брендов                       |
| **GET**| `/brand/create`   | Создать новый бренд                   |

> **Важно:** При создании товара `categoryId` и `brandId` должны соответствовать реально существующим ID в базе данных.

## 🛠 Технические особенности
- **Sequelize Sync:** Используется `{ alter: true }`, что позволяет менять структуру моделей в `mapping.js` без ручного изменения таблиц в pgAdmin.
- **JSDoc:** Все методы контроллеров задокументированы для поддержки IntelliSense в VS Code.
- **CORS:** Разрешены кросс-доменные запросы для работы с фронтендом.

## 📝 Как тестировать (test.http)
1. **Создайте категорию** (например, "Смартфоны").
2. **Проверьте её ID** через `GET /api/category/getall` (теперь это вернет JSON из базы).
3. **Создайте товар**, указав полученный `categoryId`.
